# Mafia Game
- Netty 네트워크 통신
- java gui를 이용한 인터페이스

## 기능 분석
- 회원가입
- 로그인
- 로비 - 방 만들기, 방입장, 방검색
- 대기실 - 준비기능
- 게임장 - 채팅기능, 투표기능, 직업예측 기능, 사회자 기능
- 서버 - 강제 로그아웃, 알림메세지(공지)
